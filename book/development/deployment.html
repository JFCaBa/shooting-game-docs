Deployment Process

iOS App

  1. Archive in Xcode
  2. Upload to TestFlight
  3. Submit for review

Smart Contracts

  1. Deploy token contract
npx hardhat run scripts/deploy.ts --network sepolia
  1. Verify on Etherscan
npx hardhat verify --network sepolia [CONTRACT_ADDRESS]

Server Deployment

  1. Build server
npm run build
  1. Start production server
npm start
